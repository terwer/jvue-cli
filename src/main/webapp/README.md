# webapp

## Project setup

```
yarn install
```

### Notice!!! Install plugins

```
yarn add -D file:./src/plugins/vue-cli-plugin-ssr-server
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

yarn run test
